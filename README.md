# Rummikub XL - Custom Board Inlay

This repository contains a custom 3D-printable board inlay designed specifically for Rummikub XL.

This project aims to fundamentally speed up the setup and teardown of each round. Instead of scattering unused tiles face-down on the table - which creates clutter - this system allows you to store them in dedicated, lidded holders. This keeps the game organized, speeds up drawing tiles, and prevents opponents from seeing the tiles.
